# .claude/hooks/ — 常時は薄く、必要時に厚く

`CLAUDE.md` は毎セッション丸ごと読み込まれます。だから太らせると、載せた分だけ「今から解く問題」に使えるコンテキストを先に食い、個々の指示に向く注意も薄まる。**指示は書けば効くのではなく、増やすほど一つひとつの重みが薄れる**——ここが出発点です。

このディレクトリのフックは、「常時読ませる面を薄く保ったまま、知識そのものは減らさない」ための仕掛けです。設計の全文は記事にあります → [CLAUDE.md は書くほど効かなくなる](https://zenn.dev/takotyan00/articles/claude-code-claude-md-design)。

## 3つの層に振り分ける

| 何を | どこに | いつ読むか |
|---|---|---|
| 最重要ルール・目次 | `CLAUDE.md`（60行以内） | 常時 |
| 詳細ルール・判断基準・反復手順 | `.claude/rules/*.md` | 必要時（`CLAUDE.md` のトリガーから） |
| 覚えておきたい事実（好み・判断・文脈） | `memory/*.md`（1事実1ファイル） | 関連時（下記フックが自動注入） |

`CLAUDE.md` を60行に保ち、超えた分は消すのではなく `.claude/rules/` へ出す。常時面には「いつそのルールを読むか」の1行トリガーだけ残します（トリガーの無い分割は行方不明になる）。

## memory-recall.py — 事実を「必要時だけ」引く

[`memory-recall.py`](./memory-recall.py) は `UserPromptSubmit` フックです。プロンプト送信のたびに走り、送信文と各メモリの `tags` / `description` をキーワード照合して、**関連する上位だけ**をその場で注入します。無関係な話題なら何も注入しない（コストゼロ）。素朴に tags と説明語の重なりを数えているだけで、外部依存はありません（Python 標準ライブラリのみ）。

配線（`.claude/settings.json`）:

```json
{
  "hooks": {
    "UserPromptSubmit": [
      { "hooks": [ { "type": "command",
          "command": "python3 .claude/hooks/memory-recall.py" } ] }
    ]
  }
}
```

`memory/*.md` の形式（`tags` が呼び出しの鍵）:

```markdown
---
date: 2026-07-06
tags: [料理, 段取り, 好み]
description: 平日夜の献立は「品数より一皿の満足度」を優先する
---

本文：やり取りの要点。tags と description がマッチの入口になる。
```

- **tags が命**。呼び出しはこの照合が入口なので、雑だと二度と引っかからない。日本語の別名も足すと再現率が上がる。
- 注入された内容は**参考情報**（過去の蓄積であって、その場の確定事実ではない）。断定の根拠にする前に確認する。
- `memory/**` は `.gitignore` 済み（このリポでは `README.md` だけ追跡）。生の記録はコミットされません。索引の `MEMORY.md`（常に読む厳選版）を作る場合も、機微な中身はそこに書かず1行ポインタに留めます。

## 学びを仕組みで振り分ける（自己更新ループ）

「修正を受けるたびに `CLAUDE.md` へ追記して同じミスを繰り返さない」——これを素直にやると `CLAUDE.md` が太ります。そこで追記を善意でなく仕組みにし、書く先を分散させます。

1. **検知**：セッション終了時のフックが、その回の訂正・ルール化の合図を拾って印を残す。
2. **催促**：次回開始時のフックが「未処理の学びが N 件あります」と促す。
3. **処理**：`/reflect`（コマンド／スキル）が学びを抽出し、性質ごとに保存先（全体の個人メモリ / `CLAUDE.md` / `.claude/rules/` / `memory/`）へ振り分け、**差分で提示 → 承認 → 反映**。サイレント書き換えはしない。

これで学び続けても `CLAUDE.md` だけが太る事故が起きにくくなります（追記の行き先が最初から分散しているため）。検知・催促のフックと `/reflect` の中身は運用ごとに固有なので、このリポには考え方と `memory-recall.py` の実物までを置いています。
