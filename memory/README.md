# memory/ — Tier-2 ローカルメモリ（必要時だけ引く）

セッションを跨いで覚えておいてほしい事実（好み・過去の判断・繰り返す文脈）を、**1事実1ファイル**で貯める層です。`CLAUDE.md` に書けば肥大化し、書かなければ毎回説明し直しになる——その中間として、常時は読まず、関連するときだけ `UserPromptSubmit` フック（[`.claude/hooks/memory-recall.py`](../.claude/hooks/README.md)）が自動で引きます。

## 形式

パス: `memory/YYYY-MM-DD-slug.md`

```markdown
---
date: 2026-07-06
tags: [料理, 段取り, 好み]
description: 一覧で意味が分かる一行（呼び出しの鍵）
---

やり取りの要点。tags と description がマッチの入口になる。
```

- **tags が命**：短く原子的に（`料理` `投資` `段取り` 等）。別名も足すと再現率が上がる。
- `description` は一覧で意味が分かる1行に。
- 索引 `MEMORY.md`（常に読む厳選版）は任意。作る場合も機微な中身は書かず1行ポインタに留める。

## 追跡しない

`memory/**` は `.gitignore` 済みで、この `README.md` 以外はコミットされません。生の記録はローカルに留まります。git 追跡ファイル（`persona/` など）へ昇格させるときに、実名や機微情報のマスキングを適用してください。
